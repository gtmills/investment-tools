# S&P 500 Value Stock Ranker

A tool that ranks S&P 500 stocks based on three key value investing metrics to identify potentially undervalued stocks.

## Overview

This tool analyzes stocks using a composite ranking system based on:
- **P/E Ratio** (Price-to-Earnings)
- **P/B Ratio** (Price-to-Book)
- **PEG Ratio** (Price/Earnings-to-Growth)

Lower values in these metrics typically indicate better value, so stocks are ranked with 1 being the best (lowest) value.

## How It Works

1. **Loads Data**: Reads the Excel file generated by `sp500_pe_sorter.py`
2. **Filters Stocks**: Only includes stocks with all three metrics (P/E, P/B, PEG)
3. **Ranks Each Metric**: Assigns ranks 1-N for each metric (1 = lowest/best)
4. **Calculates Average**: Computes average rank across all three metrics
5. **Sorts Results**: Orders stocks by average rank (lowest = best overall value)

## Usage

### Prerequisites

First, run the data collector:
```bash
python sp500_pe_sorter.py
```

This creates `sp500_pe_sorted.xlsx` with all the stock data.

### Run the Ranker

```bash
python value_ranker.py
```

## Understanding the Rankings

### Individual Metrics

**P/E Ratio (Price-to-Earnings)**
- Measures price paid per dollar of earnings
- Lower = cheaper relative to earnings
- Rank 1 = lowest P/E in the dataset

**P/B Ratio (Price-to-Book)**
- Compares price to book value (net assets)
- Lower = trading closer to or below asset value
- Rank 1 = lowest P/B in the dataset

**PEG Ratio (Price/Earnings-to-Growth)**
- P/E ratio adjusted for growth rate
- Accounts for future earnings potential
- PEG < 1 often considered undervalued
- Rank 1 = lowest PEG in the dataset

### Composite Score (Average Rank)

The average of the three individual ranks:
```
Avg_Rank = (PE_Rank + PB_Rank + PEG_Rank) / 3
```

**Lower average rank = Better overall value**

A stock with Avg_Rank of 10 is considered better value than one with Avg_Rank of 100.

### Important Notes

1. **Not Investment Advice**: Rankings are purely quantitative
2. **Context Matters**: Low valuations may reflect real business challenges
3. **Further Research Required**: Always investigate why a stock is "cheap"
4. **Sector Differences**: Some sectors naturally have lower valuations
5. **Missing Data**: Stocks without all three metrics are excluded

## Limitations

- Only includes stocks with positive P/E, P/B, and PEG ratios
- Does not account for qualitative factors
- Historical metrics may not predict future performance
- Sector-specific considerations not included
- Does not consider debt levels, cash flow, or other factors

## Best Practices

1. **Use as a Starting Point**: Rankings identify candidates for deeper research
2. **Check Multiple Sources**: Verify data with other financial sources
3. **Understand the Business**: Research company fundamentals
4. **Consider Sector Context**: Compare stocks within their industry
5. **Diversify**: Don't rely solely on value metrics

## Technical Details

- **Data Source**: Yahoo Finance via yfinance library
- **Ranking Method**: `rank(method='min')` - ties get the same rank
- **Filtering**: Excludes negative or zero values
- **Typical Coverage**: ~437 out of 500 S&P stocks have all three metrics

## Troubleshooting

**Error: "sp500_pe_sorted.xlsx not found"**
- Run `python sp500_pe_sorter.py` first to generate the data file

**Fewer stocks than expected**
- Some stocks lack P/E (negative earnings)
- Some stocks lack PEG (no growth data)
- Some stocks lack P/B (financial companies)
